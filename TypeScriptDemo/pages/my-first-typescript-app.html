Creating a TypeScript Project

Once you have the base tooling installed, you can make your first TypeScript project. From within Visual Studio, it's as simple as creating a new project and using the HTML Application with TypeScript template found under "Templates" -> "Other Languages" -> "TypeScript".

Starting Files

Upon creating the TypeScript project, you will have two very simple files. App.ts is the code driving the application and index.html is a simple page to load the application. (There is also an app.css file, but we're going to ignore that for this demonstration.) The default project simply renders the current date/time to the page. The pre-existing functionality isn't very interesting so we're going to improve upon it here.

Let's Iterate On That...

Now to make this more interesting we'll add some recursive functionality to this example. First, an private variable of type number is added to the MyFirstApp class. Next, instead of "Go!" just calling a Triangle constructor, it now calls a recursive function. The first iteration of this function creates the triangle we saw in the previous example. Subsequent iterations create new subtraction triangles. These triangles are supported by a new optional parameters that was added to the Triangle class constructor. If it is set to true, the triangle is rendered downwards in the color of the background.
