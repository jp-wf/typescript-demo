Interfaces - Indexers

TypeScript allows for the definition of indexable types. This allows you to define dictionary-like interfaces. Typescript supports indexable types via numeric and string indexers. However, all indexers are ultimately resolved to strings. That means defining a number index with a value of 100 is actually stored in the created type as "100". If both a numeric and string indexer are specified on an interface, the compiler requires that the resulting return types are identical.
