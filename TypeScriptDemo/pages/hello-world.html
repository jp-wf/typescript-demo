Hello World Example

This is the requisite "Hello World" example for the demonstration. The TypeScript code is defining a global namespace and exporting a single execute function. That function is simply alerting the value "Hello, world!". In the Transpiled Functionality area, that function is wired to the "Click Me" button's click event via "javascript:HelloWorld.execute()".
